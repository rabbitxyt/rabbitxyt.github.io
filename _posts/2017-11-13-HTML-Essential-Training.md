## 1. 介绍 HTML

HTML 中的内容是通过将其包装在 HTML 标签中进行标识的。你也可以通过属性来进一步增强其内容：有的属性可以告诉你关于元素更多的信息，有的属性可以提供附加的功能。

## 2. 基本页面结构

```html
<!doctype html>
<html>
  <head>
    <meta charset="utf-8">
    <meta name="description" 
          content="A page...">
    <title>Basic document</title>
  </head>
  <body>
    <h1>Page content</h1>
    <p>Page ... body tag</p>
  </body>
</html>
```
